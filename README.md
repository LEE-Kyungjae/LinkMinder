기능구현 목표

[hello.html UI에서 구현할 기능]
1.탭목록을 누르면 해당링크의 페이지가 새탭으로 열린다.
2.링크를 누르면 클립보드에 링크 주소가 복사된다.
3.비활성화 버튼을 누르면  해당 목록이 사라지고 비활성화 목록으로 이동된다.
4.현재페이지를 목록에 추가하고 제거하는 버튼을 구현한다.
